# Embedded system - 3rd Period
### Hardware reset and Mario
For embedded systems, there are 2 assignments, one is hardware reset and another is Mario. For hardware reset, it should figure out press the button. The raspberry pi Pico 2 W could be reset physically. For Mario, it is same with assignment for first period.
### Voltage meter
In Embedded system, there is an assignment, which is Voltage meter. This assignment is based on Raspberry Pi Pico 2 w and 4 digit- 7 segments display. The display shows the current outside voltage number.
### Real Time Clock
it should be finished Real time clock assignment. This assignment is based on Raspberry Pico 2 W, LCD-I2C, RTC and LM35 module. LCD-I2C shows the current time and temperature. RTC measures current time without Internet. LM35 can measure current environmental temperature.
### Dino cheater
In embedded system, there is an assignment, which name is dino cheater. In this assignment, people should use circuitpython to mock keyboard to operator dino jump. And people need to use LDR to make Dino avoid objects automatically.
## WIFI scanner & Attendance
.In embedded system. The assignment is attendance. People should use Raspberry Pico 2 W as micro-controller to control other hardware. The hardware contains RTC, LCD 2x16-I2C, Buzzer, RGB LED, mfrc522. People should figure out that when user scan the card. LCD should show the username, current time and check-in symbol. RGB LED should show the green light. Buzzer should make the sound of rising tone. And when people scan the card one more time, Buzzer should make the sound of dropping tone, RGB LED should show the orange light. LCD shows the username, current time and check out symbol. When the card doesn’t recognize, LCD shows the unknown card symbol, buzzer makes the sounds stable. RGB led shows the red light.
